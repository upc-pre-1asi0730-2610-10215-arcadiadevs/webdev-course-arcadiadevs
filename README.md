# Fundamentos de Desarrollo Web

## Resumen del Curso

Este curso de **1 hora** introduce a estudiantes de secundaria (12 a 17 años) a la creación de sitios web utilizando HTML y CSS. El enfoque es práctico, visual y accesible, diseñado para quienes no tienen experiencia previa en programación. **¡No requiere descargas ni instalaciones!**

| Información | Detalle |
|-------------|---------|
| **Duración total** | ~60 minutos |
| **Público Objetivo** | Estudiantes de 12 a 17 años sin experiencia en programación |
| **Prerrequisitos** | Ninguno |
| **Herramientas necesarias** | ¡Solo tu navegador web! (Chrome, Firefox, Safari, Edge) |
| **Plataformas** | 100% Online y gratuitas |
| **Enfoque** | HTML y CSS con ejemplos prácticos |

## Generación de Documentos PDF

El proyecto genera dos documentos PDF a partir de los archivos fuente en Markdown:

| Documento | Comando | Descripción |
|-----------|---------|-------------|
| **Plan de Curso** | `make plan` | Documento educativo sin branding ni referencias a autores |
| **Informe del Proyecto** | `make informe` | Documento con branding, guiones de video y proceso de desarrollo |

### Generar ambos documentos

```bash
make all
```

### Generar solo el plan de curso

```bash
make plan
```

### Generar solo el informe

```bash
make informe
```

Los PDFs se generan en el directorio `build/`:

- `build/upc-pre-202610-1asi0730-10215-ArcadiaDevs-course-plan.pdf`
- `build/upc-pre-202610-1asi0730-10215-ArcadiaDevs-webdev-course-report.pdf`

## Cómo Usar Este Curso

1. **Mira el video explicativo:** Accede al video de YouTube correspondiente a la lección.
2. **Abre el código inicial:** Cada clase tiene un enlace directo a CodePen con el código inicial ya precargado.
3. **Escribe y experimenta:** Acompaña la explicación escribiendo el código tú mismo.
4. **Resuelve el Reto Práctico:** Al final de cada lección encontrarás un ejercicio diseñado para evaluar lo aprendido.
