# Fundamentos de Desarrollo Web

## Resumen del Curso

Este curso de aproximadamente 1 hora introduce a estudiantes de secundaria (12 a 17 años) a la creación de sitios web utilizando HTML y CSS. El enfoque es práctico, visual y accesible, diseñado para quienes no tienen experiencia previa en programación.

**¡No requiere descargas ni instalaciones\!**
Toda la experiencia de aprendizaje se realiza mediante herramientas 100% en línea y gratuitas[cite: 216]. Solo necesitas un navegador web moderno (Chrome, Firefox, Safari o Edge).

- **Duración total**: \~60 minutos
- **Público objetivo**: Estudiantes de 12 a 17 años
- **Prerrequisitos**: Ninguno
- **Repositorio de código fuente**: [https://github.com/webdev-course-startup-name](https://www.google.com/search?q=https://github.com/webdev-course-startup-name)

## Secuencia de la lección

### Lección 1: ¿Qué es el desarrollo web? (5 minutos)

- **Descripción**: Introducción a cómo funciona el internet y la diferencia entre el contenido (HTML) y el estilo (CSS).
- **Enlace**: [Ver la lección en YouTube](https://www.google.com/search?q=https://www.youtube.com/watch%3Fv%3Dejemplo-1)
- **Práctica**: [Explorar la Web en CodePen](https://www.google.com/search?q=https://codepen.io/pen/)

### Lección 2: El esqueleto de tu página (7 minutos)

- **Descripción**: Aprende la estructura básica de una página web y cómo usar etiquetas esenciales como `<html>`, `<head>` y `<body>`.
- **Enlace**: [Ver la lección en YouTube](https://www.google.com/search?q=https://www.youtube.com/watch%3Fv%3Dejemplo-2)
- **Práctica**: [Estructura tu primera página](https://www.google.com/search?q=https://codepen.io/pen/)

### Lección 3: Organizando la información (8 minutos)

- **Descripción**: Uso de encabezados, párrafos y listas para que tu información sea clara y legible.
- **Enlace**: [Ver la lección en YouTube](https://www.google.com/search?q=https://www.youtube.com/watch%3Fv%3Dejemplo-3)
- **Práctica**: [Crea tu lista de intereses](https://jsfiddle.net/)

### Lección 4: Multimedia y navegación (8 minutos)

- **Descripción**: Cómo añadir imágenes y crear enlaces para conectar tu sitio con el resto del mundo.
- **Enlace**: [Ver la lección en YouTube](https://www.google.com/search?q=https://www.youtube.com/watch%3Fv%3Dejemplo-4)
- **Práctica**: [Inserta tu primera imagen](https://www.google.com/search?q=https://codepen.io/pen/)

### Lección 5: Conoce a CSS (7 minutos)

- **Descripción**: Introducción al diseño visual mediante selectores y propiedades básicas de color.
- **Enlace**: [Ver la lección en YouTube](https://www.google.com/search?q=https://www.youtube.com/watch%3Fv%3Dejemplo-5)
- **Práctica**: [Dale color a tu HTML](https://www.google.com/search?q=https://codepen.io/pen/)

### Lección 6: Tipografía y estilo (8 minutos)

- **Descripción**: Selección de fuentes y personalización del texto para crear una interfaz atractiva.
- **Práctica**: [Personaliza tus fuentes](https://www.google.com/search?q=https://codepen.io/pen/)
- **Enlace**: [Ver la lección en YouTube](https://www.google.com/search?q=https://www.youtube.com/watch%3Fv%3Dejemplo-6)

### Lección 7: Espacios y bordes (7 minutos)

- **Descripción**: Técnicas para centrar contenido, añadir bordes y manejar los márgenes de los elementos.
- **Enlace**: [Ver la lección en YouTube](https://www.google.com/search?q=https://www.youtube.com/watch%3Fv%3Dejemplo-7)
- **Práctica**: [Ajusta el diseño de tu página](https://www.google.com/search?q=https://codepen.io/pen/)

### Lección 8: Proyecto Final y Próximos Pasos (10 minutos)

- **Descripción**: Integración de todo lo aprendido para crear una página de perfil personal y revisión de errores comunes.
- **Enlace**: [Ver la lección en YouTube](https://www.google.com/search?q=https://www.youtube.com/watch%3Fv%3Dejemplo-8)
- **Proyecto Final**: [Crea tu Perfil Personal](https://replit.com/)
