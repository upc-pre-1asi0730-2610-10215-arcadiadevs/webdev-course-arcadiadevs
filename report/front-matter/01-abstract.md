# Resumen Ejecutivo {-}

Este documento presenta el curso "Fundamentos de Desarrollo Web", un programa de formación en desarrollo web orientado a estudiantes de secundaria (12 a 17 años) sin experiencia previa en programación. El curso tiene una duración acumulada de aproximadamente 60 minutos de contenido multimedia instructivo, estructurado en lecciones ágiles de entre 5 y 9 minutos, diseñadas para impartirse mediante una secuencia de videos de alta calidad en YouTube y complementadas con material de lectura y retos prácticos.

## Contexto y Justificación {-}

En el panorama educativo contemporáneo, el desarrollo del pensamiento computacional y la capacidad de crear contenido digital se han consolidado como competencias fundamentales para las nuevas generaciones. Aprender a desarrollar páginas web a una edad temprana no solo fomenta la creatividad y la estructuración del pensamiento lógico, sino que también democratiza el acceso a futuras oportunidades en la industria tecnológica.

HTML y CSS fueron seleccionados como base del curso debido a que son los pilares fundamentales de toda página web. HTML proporciona la estructura y el contenido, mientras que CSS controla la presentación y el diseño. Dominar estos dos lenguajes permite a los estudiantes crear sitios web funcionales y visualmente atractivos desde el primer día, sentando las bases para avanzar hacia JavaScript y otras tecnologías web modernas.

## Objetivo del Curso {-}

Introducir a los estudiantes en los fundamentos del desarrollo web mediante el aprendizaje de HTML y CSS, enfocándose en la creación de páginas web estructuradas y estilizadas. Todo esto se realiza utilizando herramientas 100% en línea y gratuitas que se ejecutan directamente en el navegador, eliminando cualquier tipo de barrera técnica de instalación o configuración local.

## Contenido Curricular Progresivo {-}

El diseño del plan de estudios se ha estructurado de forma incremental y modular, permitiendo un aprendizaje adaptativo donde cada concepto nuevo se apoya en los anteriores:

- **Módulo 1: Introducción al Desarrollo Web**
  - Concepto básico de HTML y CSS.
  - Entorno de desarrollo con CodePen.
  - Primera página web con HTML y estilos con CSS.

- **Módulo 2: Estructura y Contenido**
  - Etiquetas semánticas de HTML (headings, paragraphs, lists, links).
  - Organización de información con tablas y formularios.
  - Integración de multimedia (imágenes, audio, video).

- **Módulo 3: Estilos y Diseño**
  - Selectores, propiedades y valores en CSS.
  - Tipografía, colores y fondos.
  - Modelo de caja, márgenes, padding y bordes.

- **Módulo 4: Proyecto Final**
  - Creación de un sitio web completo aplicando todos los conceptos aprendidos.

## Enfoque Metodológico y Pedagógico {-}

El curso se rige bajo una filosofía de **"Aprender Haciendo" (Project-Based Learning)**, abordando tres pilares clave:

1. **Eliminación de fricción inicial:** Se utiliza un editor en la nube (CodePen). Esto evita que el alumno pierda interés instalando programas o configurando entornos de desarrollo, reduciendo drásticamente las barreras de entrada tecnológicas para principiantes.

2. **Enfoque visual e inmediato:** A diferencia de otros lenguajes, HTML y CSS permiten ver los resultados al instante en el navegador. Cada cambio de código se refleja inmediatamente en la página, lo que mantiene la motivación del estudiante y facilita el aprendizaje experimental.

3. **Retos y Notas de Clase:** Cada capítulo cuenta con una guía de lectura en formato markdown con ilustraciones diseñadas a medida, enlaces al código inicial/final alojados en GitHub, y una sección de "Actividades Prácticas" para incentivar la autonomía del estudiante.

## Público Objetivo y Perfil de Egreso {-}

Este curso está diseñado específicamente para estudiantes de secundaria de entre 12 y 17 años que tengan conocimientos básicos de computación a nivel de usuario y navegación web, sin ningún requerimiento de experiencia previa en codificación. Al culminar el curso, el perfil de egreso capacita al estudiante para comprender la estructura de una página web, crear contenido con HTML, estilizar con CSS, y tener las bases necesarias para avanzar hacia JavaScript y frameworks modernos.
