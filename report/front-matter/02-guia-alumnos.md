# Curso de Fundamentos de Desarrollo Web {-}

## Resumen del Curso {-}

Este curso de ~60 minutos introduce a estudiantes de secundaria a los fundamentos del desarrollo web con HTML y CSS. *No requiere descargas*, solo abrir el navegador web.

**Duración total**: ~60 minutos  
**Público Objetivo**: Estudiantes de 12 a 17 años sin experiencia en programación  
**Prerrequisitos**: Ninguno  
**Herramientas necesarias**: **Navegador web** (Chrome, Firefox, Safari, Edge, Brave)  
**Repositorio de Código Fuente**: [https://github.com/upc-pre-1asi0730-2610-10215-arcadiadevs/webdev-course-arcadiadevs](https://github.com/upc-pre-1asi0730-2610-10215-arcadiadevs/webdev-course-arcadiadevs)

## Ruta de Aprendizaje Recomendada {-}

Para aprovechar al máximo este curso, te recomendamos seguir los siguientes pasos en cada lección:

1. **Mira el video explicativo:** Accede al video de YouTube correspondiente a la lección. Presta atención a los conceptos teóricos y a cómo el instructor escribe el código.
2. **Abre el código inicial:** Cada clase tiene un enlace directo a **CodePen** con el código inicial ya precargado. Ábrelo en otra pestaña de tu navegador.
3. **Escribe y experimenta:** Acompaña la explicación escribiendo el código tú mismo. ¡No te limites a copiar y pegar! Ver los resultados al instante es la mejor forma de aprender.
4. **Resuelve el Reto Práctico:** Al final de las notas de cada lección encontrarás un pequeño ejercicio diseñado para evaluar lo aprendido. Intenta resolverlo sin ver el código de solución.

## Estructura del Repositorio {-}

El repositorio oficial del curso en GitHub está organizado para facilitarte el acceso a los recursos y archivos de trabajo de cada una de las clases:

- **`starter-files/`**: Contiene los archivos iniciales HTML y CSS para cada lección. Es el punto de partida en tu editor en línea antes de empezar a programar.
- **`completed-examples/`**: Contiene el código completamente terminado al finalizar cada clase. Utilízalo como guía de referencia si tienes algún error.
- **`report/`**: Contiene todo el material de estudio impreso o en formato digital, incluyendo las notas completas de cada clase con ilustraciones explicativas.

## Consejos para Aprender a Desarrollar Web {-}

- **No temas a los resultados inesperados:** Si tu página no se ve como esperabas, no significa que hayas hecho algo mal. El desarrollo web es un proceso de prueba y error. Revisa tu código con calma: ¿cerraste todas las etiquetas? ¿Los selectores CSS son correctos?
- **Comenta tu código:** Usa comentarios en HTML (`<!-- tu comentario -->`) para explicar qué hace cada sección. Esto te ayudará a entender tu propio código más adelante.
- **Modifica y experimenta:** Una vez que tu página funcione, intenta cambiar colores, textos, tamaños y posiciones. Es la mejor forma de entender cómo funciona CSS.
