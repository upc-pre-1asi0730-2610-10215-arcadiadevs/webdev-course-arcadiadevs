# Anexo A: Retos Prácticos {-}

## Lección 1: ¿Qué es el Desarrollo Web? {-}

**Reto del Contenido:**
Cambia el texto "Hola Mundo" por un título propio, como "Bienvenido a mi sitio" o "Mi página personal". Después cambia el párrafo "Mi primera página" por una frase corta, por ejemplo: "Estoy aprendiendo HTML y CSS" o "Aquí compartiré mis intereses". No borres las etiquetas. Cambia solo el texto que está en medio.

## Lección 2: El Esqueleto de una Página {-}

**Reto de Estructura:**
Crea una página que contenga un encabezado principal (`<h1>`), un párrafo de introducción (`<p>`), una lista de al menos 3 elementos (`<ul>` o `<ol>`), y un enlace (`<a>`) a tu sitio web favorito.

## Lección 3: Organizando la Información {-}

**Reto de Contenido:**
Crea una tabla con 3 columnas y 4 filas que muestre tu horario semanal. Añade también una lista numerada con tus 5 actividades favoritas.

## Lección 4: Multimedia y Navegación {-}

**Reto Multimedia:**
Agrega una imagen a tu página usando la etiqueta `<img>`. Asegúrate de incluir el atributo `alt` con una descripción de la imagen. Si es posible, agrega también un enlace de navegación hacia otra sección de tu página.

## Lección 5: Conoce CSS {-}

**Reto de Estilos:**
Cambia el color de fondo de tu página, modifica el color del texto del título, y ajusta el tamaño de la fuente de al menos un elemento.

## Lección 6: Tipografía y Estilo {-}

**Reto Tipográfico:**
Usa al menos 2 fuentes diferentes en tu página (una para títulos y otra para el texto). Cambia el peso de la fuente (bold, normal) y el estilo (italic) de al menos un elemento.

## Lección 7: Espacios y Bordes {-}

**Reto de Espaciado:**
Agrega márgenes y padding a tu contenedor principal. Añade un borde a tu título y redondea las esquinas usando `border-radius`.

## Lección 8: Proyecto Final {-}

**Reto Final:**
Combina todo lo aprendido para crear un sitio web personal completo que incluya:
- Un encabezado con tu nombre
- Una sección "Sobre mí" con un párrafo
- Una lista de tus intereses
- Una imagen (puede ser un placeholder)
- Estilos completos con colores, tipografía y espaciado
